# LeetCode 热题 100 核心考点手册

本参考文件汇总了 LeetCode 热题 100 的经典题目及其核心考点，用于辅助生成原创题目。

## 1. 哈希表 (Hash Table)
- **核心题目**：
  - #1 两数之和：利用哈希表实现 $O(1)$ 时间复杂度的查找。
  - #49 字母异位词分组：特征提取（排序或计数）作为哈希表的 Key。
  - #128 最长连续序列：哈希集合去重与高效查询。
- **考点要点**：空间换时间、快速查找、特征提取、频率统计。

## 2. 双指针 (Two Pointers)
- **核心题目**：
  - #283 移动零：快慢指针原地修改数组。
  - #11 盛最多水的容器：对撞指针，贪心策略缩小搜索空间。
  - #15 三数之和：排序 + 对撞指针，跳过重复元素。
  - #42 接雨水：单调性利用或左右边界记录。
- **考点要点**：对撞指针、快慢指针、单调性优化、数组区间处理。

## 3. 滑动窗口 (Sliding Window)
- **核心题目**：
  - #3 无重复字符的最长子串：维护动态窗口，记录字符最后出现位置。
  - #438 找到字符串中所有字母异位词：固定长度窗口的字符频率对比。
- **考点要点**：连续子串/子数组、窗口扩充与收缩、哈希表/计数数组辅助。

## 4. 子串/子数组 (Subarray)
- **核心题目**：
  - #560 和为 K 的子数组：前缀和 + 哈希表统计。
  - #239 滑动窗口最大值：单调队列维护窗口内最值。
  - #76 最小覆盖子串：变长滑动窗口的高级应用。
- **考点要点**：前缀和优化、单调队列、双指针窗口。

## 5. 普通数组 (Array)
- **核心题目**：
  - #53 最大子数组和：Kadane 算法（动态规划基础）。
  - #56 合并区间：排序后比较起止位置。
  - #189 轮转数组：数组翻转技巧。
  - #238 除自身以外数组的乘积：前后缀分解。
  - #41 缺失的第一个正数：原地哈希（利用下标存储信息）。
- **考点要点**：数组变换、区间合并、原地算法、前后缀。

## 6. 矩阵 (Matrix)
- **核心题目**：
  - #73 矩阵置零：利用行列首位作为标记。
  - #54 螺旋矩阵：设定边界（上下左右）模拟。
  - #48 旋转图像：转置 + 镜像翻转。
  - #240 搜索二维矩阵 II：从右上角/左下角阶梯搜索。
- **考点要点**：模拟法、坐标变换、阶梯搜索。

## 7. 链表 (Linked List)
- **核心题目**：
  - #206 反转链表：基本迭代/递归。
  - #141 环形链表：快慢指针判断。
  - #21 合并两个有序链表：归并思想。
  - #146 LRU 缓存：哈希表 + 双向链表。
  - #25 K 个一组翻转链表：复杂指针操作。
- **考点要点**：虚拟头节点、快慢指针、断链重连、多指针控制。

## 8. 二叉树 (Binary Tree)
- **核心题目**：
  - #102 层序遍历：BFS + 队列。
  - #104 二叉树的最大深度：DFS/BFS。
  - #98 验证二叉搜索树：BST 中序遍历有序性。
  - #236 最近公共祖先：递归回溯。
  - #124 二叉树中的最大路径和：树形 DP。
- **考点要点**：DFS/BFS、层级遍历、中序遍历性质、树形递归。

## 9. 图论 (Graph)
- **核心题目**：
  - #200 岛屿数量：DFS/BFS 连通分量遍历。
  - #994 腐烂的橘子：多源 BFS。
  - #207 课程表：拓扑排序（入度表 + 队列）。
  - #208 实现 Trie (前缀树)：字典树结构实现。
- **考点要点**：DFS/BFS、拓扑排序、连通性判断、前缀树。

## 10. 回溯 (Backtracking)
- **核心题目**：
  - #46 全排列：状态重置与路径选择。
  - #78 子集：二进制枚举或 DFS。
  - #17 电话号码的字母组合：递归组合。
  - #79 单词搜索：矩阵中的 DFS 回溯。
  - #51 N 皇后：约束条件与状态压缩。
- **考点要点**：决策树遍历、状态重置、剪枝优化。

## 11. 二分查找 (Binary Search)
- **核心题目**：
  - #35 搜索插入位置：标准二分查找。
  - #33 搜索旋转排序数组：分类讨论有序部分。
  - #153 寻找旋转排序数组中的最小值：局部有序性利用。
  - #4 寻找两个正序数组的中位数：复杂二分中位数寻找。
- **考点要点**：折半查找、边界处理、有序空间缩减。

## 12. 栈 (Stack)
- **核心题目**：
  - #20 有效括号：辅助栈匹配。
  - #155 最小栈：辅助栈保存当前最小值。
  - #394 字符串解码：多栈处理嵌套逻辑。
  - #739 每日温度：单调栈应用。
  - #84 柱状图中最大的矩形：单调栈计算左右边界。
- **考点要点**：辅助栈、单调栈（找左右第一个更大/更小值）。

## 13. 堆 (Heap)
- **核心题目**：
  - #215 数组中的第 K 个最大元素：维护小顶堆。
  - #347 前 K 个高频元素：桶排序或堆。
  - #295 数据流的中位数：对顶堆。
- **考点要点**：Top K 问题、动态最值维护。

## 14. 贪心算法 (Greedy)
- **核心题目**：
  - #121 买卖股票的最佳时机：局部最小值更新。
  - #55 跳跃游戏：维护当前最远可达位置。
  - #763 划分字母区间：统计字符最后出现位置。
- **考点要点**：局部最优、最远/最佳状态维护。

## 15. 动态规划 (Dynamic Programming)
- **核心题目**：
  - #70 爬楼梯：简单线性 DP。
  - #198 打家劫舍：状态转移方程设计。
  - #322 零钱兑换：完全背包变体。
  - #300 最长递增子序列：$O(N^2)$ 或 $O(N \log N)$ 优化。
  - #72 编辑距离：二维 DP 状态转移。
  - #5 最长回文子串：区间 DP 或中心扩展。
- **考点要点**：状态定义、转移方程、Base Case、空间优化。
